# EDUCADO Training Course: SLURM

Welcome to this workshop on using Slurm for High Performance Computing (HPC).
This workshop is designed to take you from running scripts on your local machine to scaling them up on a Slurm-managed cluster.

- [Start Presentation](https://bernddoser.github.io/EDUCADO_Slurm_Workshop)

## Prerequisites

- Access to a Slurm cluster (for the hands-on part).


## Getting Started

Clone this repository to your cluster home directory:

```bash
git clone https://github.com/BerndDoser/EDUCADO_Slurm_Workshop.git
cd EDUCADO_Slurm_Workshop/hands-on
```
